About Notes Management System

The Notes Management System is a secure web application built using Flask (Python) and MySQL.

It allows users to efficiently create, manage, and organize their notes with complete privacy. Each user gets a dedicated workspace ensuring secure access.

Features
  • User Registration & Login
  • Secure Password Encryption
  • Create, Read, Update, Delete Notes
  • Forgot Password with OTP
  • User-specific data access
Technologies Used
  • Frontend: HTML, CSS, Bootstrap
  • Backend: Flask (Python)
  • Database: MySQL
